## Curso de Data Science

Este repositório reúne os materiais, exercícios, projetos e recursos utilizados ao longo do curso de Data Science.

## Processo de Avaliação

Os estudantes podem optar por avaliação contínua ou avaliação por exame final.

Avaliação contínua:

* 3 mini-quizzes — 40%
* 1 projeto de grupo — 60%

Avaliação por exame:

* Exame final — 100%

Os quizzes e o exame podem incluir questões práticas, envolvendo análise de dados, programação ou interpretação de resultados.

## Estrutura do Repositório

`00_Datasets`

Pasta onde deverão estar contidos os datasets utilizados ao longo do curso.


`Practical`

Pasta com o conteúdo prático, incluindo notebooks Python e exercícios.

`Theory`

Pasta com o conteúdo teórico e materiais de suporte ao curso.

Este repositório será atualizado ao longo do curso conforme novos conteúdos forem disponibilizados.
